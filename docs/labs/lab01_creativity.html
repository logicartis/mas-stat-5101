Plotting Data

Create a dot plot

  • Center all dots on x-axis value 1

  • y is the score variable

  • binaxis is the axis to bin along, using y here

  • stackdir specifies direction in which dots are stacked

ggplot(data = creativity, mapping = aes(x = 1, y = score)) +
    geom_dotplot(binwidth = 1, binaxis = "y", stackdir = "center")

Create dot plots by treatment group

  • Use the treatment column in the x-axis
ggplot(data = creativity, mapping = aes(x = treatment, y = score)) +
    geom_dotplot(binwidth = 1, binaxis = "y", stackdir = "center")

Avoid redundancy by building a template and adding different geom layers to the template

  • Create a template
creativity_setup1 <- ggplot(data = creativity, mapping = aes(x = score))
  • Add geom layer - histogram
creativity_setup1 + geom_histogram()
`stat_bin()` using `bins = 30`. Pick better value `binwidth`.
  • Add geom layer box_plot
creativity_setup1 + geom_boxplot()
  • Flip the box plot x and y axes
creativity_setup1 + 
    coord_flip() +
    geom_boxplot()

Work with treatment group

creativity_setup2 <- ggplot(data = creativity, mapping = aes(x = treatment, y = score))
  • Draw box plot per treatment group
creativity_setup2 + geom_boxplot()
  • Change theme to classic theme and redraw the same box plot
creativity_setup2 + 
    geom_boxplot() +
    theme_classic()

Specify different colors to visually separate groups

  • Create the template
creativity_setup3 <- ggplot(
    data = creativity, 
    mapping = aes(
        x = score,
        fill = treatment,
        color = treatment
    )
)
  • Draw histogram
creativity_setup3 + 
    geom_histogram(position = "dodge")
`stat_bin()` using `bins = 30`. Pick better value `binwidth`.
  • Change color scale
creativity_setup3 +
    scale_color_grey() + 
    scale_fill_grey() +
    geom_histogram(position = "dodge")
`stat_bin()` using `bins = 30`. Pick better value `binwidth`.

ggsave("figure.png")

Self-Assessment

Read tomato.csv

tomato_df <- read.csv("../data/tomato.csv")
names(tomato_df)
[1] "treatment" "yield"

View the first 6 records

head(tomato_df)
  treatment yield
1         a  0.39
2         b  0.44
3         a  1.83
4         a  5.66
5         b  0.33
6         a  8.06

Draw side-by-side box plots of each yield for each of the two fertilizers

ggplot(data = tomato_df, mapping = aes(x = treatment, y = yield)) +
    geom_boxplot() +
    theme_classic()

Estimate the difference in mean yield between the two fertilizers (a - b). Include units.

mean_a = tomato_df |> filter(treatment == "a") |> pull(yield) |> mean()
mean_b = tomato_df |> filter(treatment == "b") |> pull(yield) |> mean()
paste("Mean difference: ", round(mean_a - mean_b, 2), "pounds")
[1] "Mean difference:  6.53 pounds"

Estimate the difference between in median yield the two fertilizers (a - b). Include units.

median_a = tomato_df |> filter(treatment == "a") |> pull(yield) |> median()
median_b = tomato_df |> filter(treatment == "b") |> pull(yield) |> median()
paste("Median difference: ", round(median_a - median_b, 2), "pounds")
[1] "Median difference:  4.64 pounds"
